# vehicles-project
Este proyecto es un dashboard interactivo en Streamlit que permite analizar datos de coches de manera sencilla y visual. El dashboard incluye filtros, tablas y gráficos para explorar el conjunto de datos de manera interactiva.

Propósito

El objetivo de este proyecto es practicar análisis exploratorio de datos y la creación de aplicaciones web interactivas con Streamlit, permitiendo visualizar información de precios y modelos de coches mediante gráficos dinámicos.

Cómo usarlo
Filtra los datos según:
Marca
Modelo
Año
Visualizaciones disponibles:
Histograma de precios: muestra la distribución de precios.
Scatter plot (gráfico de dispersión): relaciona precio con año.
Explora la tabla de datos para ver todos los registros filtrados.
Dashboard en línea

Puedes ver la aplicación web aquí: Dashboard

Requisitos

Para correr el proyecto localmente necesitas instalar:

pandas
streamlit
plotly-express

Ejemplo de cómo instalar las librerías:

pip install pandas streamlit plotly-express
Archivos incluidos
app.py → Archivo principal de la aplicación web.
cars.csv → Conjunto de datos de los coches.
requirements.txt → Librerías necesarias para ejecutar la app.
README.md → Este archivo con instrucciones y detalles del proyecto.
